
一. 块级元素和内嵌元素的区
=====================================
1. 块级元素：p,div,ul,ol,li,h1-h6等
{
	特点：默认都占一行
	支持设置宽高
	高度自适应（在不设置高度的情况下，自适应内容）
	支持所有css样式
	块级元素可以嵌套大多数的块级元素和内嵌元素
	特殊的块级元素p h1-h6 dt dl dd不可以嵌套块级元素（其实嵌套也没关系，最好不）

}

2.内嵌元素（行元素）
{
	特点：可以同行显示
	内容撑开宽高
	不支持设置宽高
	有些css样式不支持（margin）
	内嵌元素可以嵌套内嵌元素，不可以嵌套行内元素
}
